Wiki-XML-Creator by kimhauritz

Create WikiMedia XML files from books in html, text and pdf (via pdftohtml) format.

Wiki XML Creator is a command line tool for the WikiReader from openmoko, the aim is to generate a WikiMedia XML file from books in html, txt and pdf format that can be converted into the WikiReader's dataformat (by using their software). This application will enable users to read their favorite book from project gutenberg and other free sources, on the WikiReader.

Dependencies

The application is a command line application, and is written in Java using the eclipse framework.

Install

This Application is still in the prototype state so: Download > Fix Bugs > Compile > Have Fun
